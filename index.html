Parte 3

Inspecionar elementos e testar mudancas no DevTools.

2. O esqueleto basico do HTML

Todo documento HTML comeca com uma estrutura minima. Ela informa ao navegador que tipo de documento esta sendo lido e onde fica o conteudo visivel da pagina.

<!DOCTYPE html> define que o documento usa HTML5.
<html> envolve toda a pagina.
<head> guarda configuracoes e metadados.
<body> contem tudo o que aparece para o usuario.
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha primeira pagina</title>
</head>
<body>
    <h1>Ola, mundo!</h1>
    <p>Esse e meu primeiro conteudo em HTML.</p>
</body>
</html>

3. O que existe dentro de head e body

Uma duvida comum de quem esta comecando e separar o que fica no <head> do que fica no <body>. O head prepara a pagina. O body entrega o conteudo.

Head

  • <meta charset> define a codificacao de caracteres.
  • <meta name="viewport"> melhora a exibicao em celular.
  • <title> muda o nome exibido na aba do navegador.
  • <link> conecta um arquivo CSS externo.

Body

  • <h1> a <h6> criam titulos.
  • <p> cria paragrafos.
  • <a> cria links.
  • <img>, <ul>, <div> e outras tags montam a pagina.

5. Tags basicas para praticar

Essas sao algumas das tags mais simples para demonstrar ao vivo em sala. Elas ajudam a mostrar como o HTML estrutura a informacao antes mesmo de pensar em design.

Texto e organizacao

  • <h1> a <h3> para hierarquia de titulos.
  • <p> para paragrafos.
  • <br> para quebra de linha.
  • <hr> para separar assuntos.

Conteudo e navegacao

  • <a> para links.
  • <img> para imagens.
  • <ul> e <ol> para listas.
  • <div> para agrupar blocos.
<h2>Lista de materiais</h2>
<ul>
    <li>Notebook</li>
    <li>Navegador</li>
    <li>Editor de codigo</li>
</ul>

7. Usando o DevTools do navegador

O DevTools permite inspecionar a pagina, localizar elementos e testar alteracoes sem precisar salvar o arquivo toda vez. E uma ferramenta importante para aprender e para depurar erros.

Inspecionar

Clique com o botao direito em um elemento e escolha Inspecionar.

Editar HTML

No painel Elements, e possivel mudar textos, classes e atributos.

Testar CSS

No painel Styles, podemos ligar e desligar regras para entender o efeito visual.

Exemplos de uso em aula:
- alterar o texto de um <h1>
- remover uma tag temporariamente
- testar uma cor no CSS
- verificar se uma imagem possui atributo alt

8. Roteiro sugerido para demonstracao

Para manter a aula dinamica, podemos seguir uma sequencia curta que mistura explicacao e pratica. Assim o publico acompanha a teoria enquanto observa o codigo sendo manipulado.

  1. Mostrar a estrutura minima de um arquivo HTML.
  2. Explicar o papel de head e body.
  3. Criar um titulo e um paragrafo ao vivo.
  4. Adicionar um link e uma lista.
  5. Abrir o DevTools e editar o HTML pela aba Elements.
  6. Testar mudancas de estilo na aba Styles.

9. Conclusao

A base do HTML esta em entender a estrutura do documento e praticar pequenas alteracoes de codigo. Quando combinamos isso com o DevTools, o aprendizado fica mais visual, rapido e facil de demonstrar.

Resumo da equipe 1: apresentar o esqueleto do HTML, ensinar tags basicas, manipular exemplos ao vivo e usar o DevTools para reforcar a pratica.
