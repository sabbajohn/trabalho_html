# Sobre o Projeto
Este projeto contem o material da Equipe 1 para a apresentacao de HTML.

A proposta foi padronizada com o material da Equipe 2, mantendo o formato de pagina unica com secoes explicativas, exemplos visuais e blocos de codigo.

# Estrutura da Apresentacao
A apresentacao foi organizada em formato de aula:

* introducao ao esqueleto basico do HTML
* explicacao das areas `head` e `body`
* demonstracao de tags basicas
* parte pratica com manipulacao de codigo
* uso do DevTools do navegador para inspecao e testes

# Objetivo
Ensinar os fundamentos de HTML de forma didatica, visual e pratica, ajudando a turma a entender a estrutura da pagina e como pequenas alteracoes no codigo afetam o resultado final no navegador.
